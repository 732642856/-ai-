## 二、15 个功能缺口 × 开源方案

### Phase 1：角色控制（4 个缺口）

| # | 功能 | 开源方案 | 复用方式 | 许可证 |
|:-:|------|---------|---------|:------:|
| 1 | **三视图生成** | `MV-Adapter` / `CharacterGen` | 方案A：通过中转站 API + prompt 工程（已实现 route.ts） | — |
| 2 | **多角度控制** | `react-three-fiber` + `pose-avatar` | 用 Three.js 控制 3D 角色朝向 → 截图出 2D 图 | MIT |
| 3 | **姿势参考（火柴人）** | `LaichuLai/openpose-skeleton-editor` ⬅️ **直接复制** PoseEditor.jsx | 提取 BODY_25 关键点 + Canvas 拖拽逻辑，封装成独立组件 | MIT |
| 4 | **焦点编辑（局部精修）** | `3rChuss/react-canvas-masker`（npm） ⬅️ **npm install** | 自由绘制蒙版 → 调用中转站图片编辑 API（/v1/images/edits） | MIT |
