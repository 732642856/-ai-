## 二、画布/节点系统对比

### 2.1 画布形态

| 特性 | TapNow | 小云雀 | LibTV | **StarCanvas 当前** |
|------|--------|--------|-------|---------------------|
| 画布类型 | 无限画布 | 资产创作画布 | 无限画布 + 节点工作流 | ✅ 无限画布（ReactFlow） |
| 节点类型 | 文本/图片/视频/音频/模型 | 角色/场景/文本/图片/视频/音频（6类） | 文本/图片/视频/音频/脚本（5 类） | ✅ 30 种 nodeKind |
| 节点连线 | ✅ 拉线连接 | ❌ 资产间 @调用 | ✅ 自由连线搭建工作流 | ✅ 多边类型 |
| 画布共享 | ✅ TapTV 克隆 | ❌ | ✅ 画布分享 + 社区展示 | ❌ 无 |
| 工作流保存 | ✅ 一键克隆/复用 | ❌ | ✅ 打组保存/一键重复执行 | ❌ 无 |

### 2.2 StarCanvas 差距

- **✅ 领先**：30 种 nodeKind 远超三者（TapNow ~5, 小云雀 6, LibTV 5）
- **❌ 缺失**：画布共享/工作流保存/克隆机制，三者都有
- **❌ 缺失**：节点连线在 StarCanvas 仅是 UI 展示，三者均用作工作流数据流

## 三、AI Agent 能力对比

这是三者与 StarCanvas 差距最大的维度。

### 3.1 TapNow Agent

| 能力 | 说明 |
|------|------|
| 对话式 AI 助手 | 画布右侧对话框，基于灵感生成分镜、获取参考图 |
| 提示词反推 | 上传参考图，自动生成对应提示词 |
| 智能参考推荐 | 搜索动作/构图/风格参考图 |
| 音乐提示词生成 | 根据分镜图自动编写 BGM 提示词 |

**Agent 定位**：辅助型——帮人写提示词、找参考，不代替人执行多步操作。

### 3.2 小云雀短剧 Agent 2.0

| 能力 | 说明 |
|------|------|
| 智能托管 | 全流程自动化：剧本→资产→分镜→视频 |
| AI 生剧本 | 一句话 → 100 集剧本 + 人物小传 + 分镜剧本 |
| 剧本智能解析 | 上传剧本 → 自动提取人物/场景 → 生成资产节点 |
| 资产自动同步 | 修改角色 → 全局自动更新 |
| @调用资产 | 分镜生成环节一键调用素材库预设资产 |

**Agent 定位**：执行型——全流程托管，人负责创意方向，Agent 负责落地。

### 3.3 LibTV Agent

| 能力 | 说明 |
|------|------|
| OpenClaw Skill 包 | 官方提供 `libtv-skills`，Agent 可调用全部功能 |
| 全自动视频生产 | 一句话 Prompt → Agent 完成剧本/分镜/视频/剪辑 |
| 爆款复刻 | 给 URL → Agent 解析镜头语言 → 一键复刻 |
| 音乐 MV 自动生成 | 上传音乐 → Agent 生成匹配视觉叙事 |
| 多 Agent 生态 | 支持 OpenClaw/Autoclaw/Kimi Claw/MaxClaw/QClaw |
| 5 分钟+长视频 | Agent 可生成超过 5 分钟的完整内容 |

**Agent 定位**：平台型——"Creator + Agent 双入口"，既服务人也服务 AI。

### 3.4 StarCanvas 差距

| 维度 | StarCanvas 当前 |
|------|----------------|
| 画布级 Agent | ❌ 无——Canvas Agent 仅做节点操作，不做内容创作全流程自动化 |
| 剧本→分镜→视频托管 | ❌ 无 |
| Agent 协议/接口 | ❌ 无——未对外开放 Agent 调用 |
| 提示词反推/参考推荐 | ❌ 无 |
| 资产自动同步/全局更新 | ❌ 无 |

## 九、建议的增强优先级

基于竞争分析，StarCanvas 应在保持差异化优势的前提下补上关键短板：

### P0 — 即时（决定产品是否可用）
1. **AI 内容创作 Agent 全流程托管** — 参考小云雀的智能托管模式，基于现有 StoryboardDirectorAgent 扩展
2. **剧本→分镜→视频自动化管线** — 七步链路已有骨架，需补肉

### P1 — 近期（决定产品竞争力）
3. **画布工作流保存/加载/克隆** — 参考 TapNow 的克隆机制
4. **角色资产 + 一致性管理** — 参考小云雀的资产联动
5. **主流视频模型集成** — 接入可灵/Seedance 等

### P2 — 中期（专业度提升）
6. **分镜专业化工具** — 多角度/焦点编辑/首尾帧
7. **Agent 开放协议** — 参考 LibTV 的 Skill 包设计
8. **音频系统** — TTS + AI 音乐

### P3 — 远期（生态建设）
9. **社区 + 工作流市场** — 参考 TapTV
